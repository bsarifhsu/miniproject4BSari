Welcome to Inventory App

The Inventory App is a web application developed using Django, a Python web framework. It serves as a simple yet efficient solution for managing and tracking product inventory. Users can log in to the app, view a list of products, and perform actions such as editing and deleting products. The app also includes user authentication, allowing registered users to log in, while new users can sign up.
